# virtual-network-lab
A hands-on IT support and networking lab built using:

- Ubuntu Linux
- Windows 11
- VirtualBox
- Wireshark

## Goals
- Learn networking fundamentals
- Practice troubleshooting
- Analyze traffic using Wireshark
- Simulate real-world IT environments
- Build practical IT support experience

## Lab Structure

```bash
Virtual-Network-Lab/
├── screenshots/
├── notes/
└── configs/
```
